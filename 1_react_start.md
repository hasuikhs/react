# 1. React Start

## 0. 사전 작업

- CRA(Create React App) 을 사용하면 configuration을 안해도 되므로 편리

```bash
# npm
$ npm install -g create-react-app

# yarn
$ yarn add global create-react-app
```

## 1. 프로젝트 생성과 시작

- 프로젝트명은 camelCase 작성 불가능

  ```bash
  $ create-react-app project_name
  ```

- 프로젝트 시작은

  ```bash
  # npm
  $ npm start
  
  # yarn
  $ yarn start
  ```
